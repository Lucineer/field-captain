# Field Captain

> CLI toolkit for managing Deckboss deployments, technician workflows, and fleet operations in the field.

The technician's command-line companion for maintaining the bridge between technology and the human domain.

## Install

```bash
pip install field-captain
```

## Commands

```bash
field-captain init --vessel FV-Pinnacle --port dutch-harbor
field-captain install --checklist marine-standard
field-captain report --type installation --output pdf
field-captain health --unit FV-Pinnacle --remote
field-captain fleet update --profile lucineer/marine
```

## The Technician Paradigm

Physical AI systems need a human who understands both technology and the domain. Field Captain is that human's tool.

See [The Technician](https://github.com/Lucineer/the-technician) for the full paradigm.
